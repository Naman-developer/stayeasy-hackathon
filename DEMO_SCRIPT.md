## 10-Step Live Flow
1. **Landing page**
   - Show hero, quick search, categories, featured listings, trust badges.
2. **Login as owner**
   - Open owner dashboard and add a new property.
3. **Pending state**
   - Show listing status = `pending` on owner dashboard.
4. **Switch to admin**
   - Open admin dashboard pending table.
5. **Approve listing**
   - Click Approve. Mention this is the critical moderation workflow.
6. **Switch to student/tenant**
   - Search listing in public search and show it appears after approval.
7. **Booking + payment**
   - Open property details -> Book Now -> Create booking -> Pay Now (mock success).
8. **Worker booking**
   - Open helper marketplace, select worker, create helper booking.
9. **Hostel + parent monitoring**
   - Briefly show hostel dashboard (attendance/outpass) and parent dashboard (approval view).
10. **SOS + admin visibility**
   - Trigger SOS from student dashboard and show notification/admin monitoring signal.

## Key One-Liners While Demoing
- "Only approved listings go live in search."
- "One platform for accommodation + helpers + safety workflows."
- "Built as a modular MVP with clean upgrade path."
